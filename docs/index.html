Brandon Allen

2022-02-01

1 Summary

The Alberta Biodiversity Monitoring Institute (ABMI) has developed a monitoring program for lichens, mosses, vascular plants, and soil mites. This technical document contains information describing how data was collected, processed, and analysed. This document will be updated as necessary. Readers can find the most up-to-date results on the ABMI Biodiversity Browser (https://www.abmi.ca/home/data-analytics/biobrowser-home).

1.1 Acknowledgement

We would like to acknowledge that this work would not be possible without the dedication of the past and present ABMI staff. Without our field staff, geospatial experts, and ecologists, we would not be able to perform this work.
